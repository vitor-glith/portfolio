PROJETO FRONT-END

Nike e-commerce UI

Um redesign moderno e interativo de uma loja online inspirada na Nike e experiência de usuário .

Design responsivo · Interface moderna

Sobre o projeto

Este projeto foi desenvolvido como parte do meu portfólio para demonstrar habilidades em front-end. O objetivo era criar uma interface visualmente atraente inspirada no estilo minimalista e moderno da Nike, e uma experiência de usuário fluida.

O design foi pensado para ser responsivo, funcionando perfeitamente em dispositivos móveis, tablets e desktops. Utilizei técnicas de CSS como Grid e HTML para criar uma navegação intuitiva e agradável.

Desafios & Aprendizados

Durante o desenvolvimento deste projeto, enfrentei desafios como os arquivos sendo perdidos por terem formatado meu computador no meio do processo e o progresso foi todo perdido.

Este projeto me permitiu aprimorar minhas habilidades em CSS moderno, especialmente no uso de Grid Layout para criar layouts complexos. Além disso, pratiquei com HTML puro para criar interatividade com mais facilidade.

Próximos passos

Gostei de desenvolver este projeto e pretendo seguir nessa parte do design gráfico, UX e UI, talvez algum dia eu termine este site, mas por enquanto vou manter ele simples assim.
